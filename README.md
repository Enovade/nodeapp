# A Simple API app based on Node.js and Express

# Guide to install this app
## Clone this app
```
$ git clone https://github.com/Hanafiah-Enovade/nodeapp.git
```

## Install the app
```
$ cd nodeapp
$ npm install
```

## Run the app
```
$ node index.js
```

## Test the app
http://localhost:4000

http://localhost:4000/about
